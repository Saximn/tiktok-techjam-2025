# Training Scripts

This directory contains all machine learning training scripts for the VoiceShield project.

## Scripts Overview

### Core Training Scripts
- `master_training.py` - Main training orchestrator
- `simple_fine_tuning.py` - Basic fine-tuning implementation
- `production_fine_tuning.py` - Production-ready fine-tuning

### Advanced Training Scripts
- `advanced_audio_training.py` - Advanced audio processing training
- `advanced_transformer_training.py` - Transformer architecture training
- `enhanced_sota_training.py` - Enhanced state-of-the-art training
- `fixed_sota_training.py` - Fixed SOTA training implementation

### Specialized Training Scripts
- `fine_tune_voice_models.py` - Voice model fine-tuning
- `fine_tune_with_real_data.py` - Real dataset fine-tuning
- `real_data_fine_tuning.py` - Real-world data training
- `sota_audio_training.py` - State-of-the-art audio training
- `sota_fine_tuning.py` - SOTA fine-tuning implementation

## Usage

Each script can be run independently with appropriate configuration files.
Check the main project documentation for detailed usage instructions.

# Server Components

This directory contains all server-side components for the VoiceShield project.

## Server Files

### Main Servers
- `main.py` - Primary Python server application
- `python_server.py` - Python backend server
- `dev_server.js` - Development JavaScript server

## Usage

### Python Server
```bash
python main.py
```

### Development Server
```bash
node dev_server.js
```

## Configuration

Server configuration files are located in the `../config/` directory.

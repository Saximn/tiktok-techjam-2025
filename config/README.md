# Configuration Files

This directory contains all configuration files for the VoiceShield project.

## Configuration Files

### Package Management
- `package.json` - Node.js dependencies and scripts
- `package-lock.json` - Locked dependency versions
- `requirements.txt` - Python dependencies
- `requirements_working.txt` - Working Python dependencies snapshot

### Build Configuration
- `vite.config.js` - Vite build tool configuration
- `webpack.config.js` - Webpack bundler configuration

## Installation

### Node.js Dependencies
```bash
npm install
```

### Python Dependencies
```bash
pip install -r requirements.txt
```

## Build Commands

Check `package.json` scripts section for available build commands.

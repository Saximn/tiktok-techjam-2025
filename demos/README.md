# Demo Scripts

This directory contains all demonstration scripts for the VoiceShield project.

## Available Demos

### Comprehensive Demos
- `COMPREHENSIVE_DEMO.py` - Full feature demonstration
- `COMPREHENSIVE_DEMO_FINAL.py` - Final comprehensive demo version
- `demo_final.py` - Complete demo implementation

### Specialized Demos
- `LIVE_VOICE_DEMO.py` - Live voice processing demonstration
- `PRODUCTION_DEMO.py` - Production-ready demo
- `advanced_pii_demo.py` - Advanced PII protection demo
- `quick_demo_test.py` - Quick functionality test demo

## Usage

Run any demo script to see VoiceShield in action:

```bash
python LIVE_VOICE_DEMO.py
```

Each demo showcases different aspects of the voice privacy protection system.
